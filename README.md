# rest_api_mcp_tools_generator
This Code a process a swagger interface of the Rest API service and will generate a MCP Server for Rest API Service.

This code base has 2 main component:

1. Tools Generator : This component takes a swagger url as input, parse all the available APIs in the swagger and generates list of all APIs with it's description, input parameters and output parameters in a specified format (YAML).

    Input for the Component:
    1. Rest APIs swagger URL
    2. Rest API URL
    3. Authentication Mechanism
    4. Description of the Rest API service
    5. Rest API description and it's input and output parameter description

    Output of the Component:
    1. YAML files for the rest APIs tools with details like
        i. Rest API URL
        ii. Authentication Mechanizsm
        iii tools list with all input and output parameters

    Feature provided by the Tools:
    1. UI to test swagger URL, Rest API URL and Authentication mechanism as input.
    2. Read all the rest apis details from the swagger interface.
    3. Provide user option to enhance the description of rest api, description of input parameters and output parameters.

2. MCP Server for the the Tools : This codebase uses the YAML generated by the 1st tool and servers as a MCP server for the tools.
